Synchronize Fields (Vue.js)

Back to index

Synchronize field values with text inputted by the user.

The text of the <span> elements of the "top" section, display the value of the fields.

Check Vue.js documentation: https://vuejs.org/guide/essentials/forms.html

Use the v-model directive

Note: The behavior of the fields Name and Grade are different. Apply the modifier lazy to one of the v-model directives

Output
{{ name }} {{ grade }} {{ period }}
