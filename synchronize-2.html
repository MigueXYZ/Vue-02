Synchronize field values with text inputted by the user.

The text of the <span> elements of the "top" section, display the value of the fields.

Check Vue.js documentation for binding select options to an array of objects https://vuejs.org/guide/essentials/forms.html#select and https://vuejs.org/api/composition-api-lifecycle.html for the onMounted event

Note: In this version, the options of the select field will be binded to an array of objects

Note: Use the v-for directive to fill the options of the select field

Suggestion: Bind the options value to an object instead of binding it to a string value.

Output
{{ name }} {{ grade }} {{ selectedPeriod.text }} ({{ selectedPeriod.value }})
